# pokedex

# Installation

  1.Cloner le repo
  
  Prérequis
  
    - Avoir python d'installé sur ca machine
    - Avoir le framework Django ([https://docs.djangoproject.com/fr/4.0/howto/windows/])
    
 2.Lancer le projet
 
    Liste des commandes:
    
    Python  manage.py  runserver
    
    python manage.py tailwind start
